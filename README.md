## Introduction
Upcoming project :)
## Installation
1. Clone this project
```bash
git clone https://github.com/ItzTheLT/WholesomeSquad-Discord.js
```
2. Open CMD
```cmd
cd wholesomesquad && npm install
```
3. Open config.json file and edit
4. Run the bot and enjoy :)
```cmd
node bot.js
```
